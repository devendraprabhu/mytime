# User Data Collection System

This project is a backend application designed to connect to **MongoDB** and securely collect and store user details. It serves as a foundation for user management systems, registration forms, or onboarding flows.

* **Live Database Connection:** Robust integration with MongoDB Atlas/Local via Mongoose.

* **REST API:** Simple endpoints to POST user data and GET stored records.
